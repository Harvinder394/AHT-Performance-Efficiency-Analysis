# 📊 AHT Performance & Efficiency Analysis

## 🔍 Problem Statement

A recent process update increased operational complexity, leading to a rise in Average Handle Time (AHT). The objective was to analyze performance across queues, locations, and agent experience to identify inefficiencies and improvement areas.

---

## 🎯 Objective

* Analyze AHT performance against target (15 mins)
* Identify high-impact problem areas
* Understand drivers of increased AHT
* Recommend actionable improvements

---

## 📊 Key Insights

* Queue 4 identified as critical with AHT ~30% above target
* Queue 2 showing early signs of performance degradation
* New agents contributing to higher AHT → training gap
* Location impact minimal → issue is process-driven

---

## 🎯 Recommendations

* Conduct RCA for Queue 4 and optimize process steps
* Implement targeted coaching for high AHT agents
* Replicate best practices from efficient queues
* Introduce structured ramp-up program for new agents
